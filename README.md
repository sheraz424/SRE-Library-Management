# 📚 Library Management System (Java + MySQL + SonarQube)

## 📌 Project Overview
The Library Management System is a desktop-based application developed in Java to manage core library operations such as book issuance, returns, borrower management, and administrative control. The system follows an object-oriented design and integrates a MySQL database for persistent data storage. Code quality is analyzed using SonarQube.

---

## 🧰 Tech Stack
- Java (Core OOP)
- MySQL Database
- JDBC (Database Connectivity)
- SonarQube (Code Quality Analysis)
- Draw.io (UML Diagrams)
- Git & GitHub (Version Control)

---

## ⚙️ Features

### 👤 User Roles
- Borrower
- Clerk
- Librarian
- Admin

### 📖 Core Functionalities
- Book search and filtering
- Book issuance and return system
- Hold request system for books
- Fine calculation for overdue books
- Borrower profile management
- Book inventory management
- Administrative control panel

---

## 🏗️ System Architecture
The system follows a layered architecture:

- **Presentation Layer:** Console-based UI
- **Business Logic Layer:** Java OOP classes (Library, Book, Person, etc.)
- **Data Layer:** MySQL Database via JDBC

---

## 🧪 Code Quality Analysis
The project is analyzed using SonarQube, which highlights:
- Code smells
- Bugs and vulnerabilities
- Maintainability rating
- Security hotspots
- Duplication issues

> Note: The system is academic in nature and focuses on functionality and learning OOP concepts.
